# Homer

> *"It's not easy to juggle a pregnant wife and a troubled child, but somehow I managed to fit in eight hours of TV a day."* —Homer Simpson

A Chrome extension that hides distractions on YouTube so you can stay focused.

## Features

| Toggle | Default | Description |
|---|---|---|
| Hide Shorts | On | Removes Shorts shelves and sidebar nav link |
| Block Shorts Page | On | Blocks `/shorts` pages entirely |
| Hide Sidebar Recommendations | On | Removes the "Up next" sidebar on video pages |
| Hide Topic Chips | On | Removes filter chips at the top of the home page |
| Hide Explore Panels | On | Removes "Explore more topics" sections |
| Hide Loading Grid | On | Removes ghost grid placeholders during page load |
| Hide End Cards | Off | Removes video end screen cards and overlays |
| Hide Comments | Off | Removes the comment section |

All toggles are accessible from the extension popup. Settings sync across devices and apply instantly — no page reload needed.

## Install

1. Clone this repo or download as ZIP
2. Open `chrome://extensions/` in Chrome
3. Enable **Developer mode** (top right)
4. Click **Load unpacked** and select the project folder
